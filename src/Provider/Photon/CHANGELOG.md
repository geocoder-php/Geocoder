# Change Log

The change log describes what is "Added", "Removed", "Changed" or "Fixed" between each release.

### Removed

- Drop support for PHP 7.3

- Migrate from PHP-HTTP to PSR-18 client

- Add name proprety
- Add country code property

### Added

### Removed

- Drop support for PHP 7.2
